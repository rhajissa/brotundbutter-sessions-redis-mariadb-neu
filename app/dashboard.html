ℹ️ Systemhinweise

Scanner-Simulation: Verwenden Sie das Eingabefeld wie einen Barcode-Scanner. Tippen Sie einen Code ein und drücken Sie Enter.

Ausleihe: Zuerst muss die Mitgliedskarte eingelesen werden, bevor Bücher gescannt werden können. Erst nach Klick auf "Ausleihe bestätigen" wird die Transaktion persistiert.

Rückgabe: Zeigt direkt den Ausleihstatus, den Leser und eventuelle Überschreitungen der Leihfrist an.

Virtueller Belegdruck: Beide Aktionen schieben einen Druckauftrag in eine asynchrone Redis-Warteschlange. Unser PrintDaemon druckt die Belege in den Ordner printed_receipts/.
